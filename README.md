# NYC Flights 2023 — Interactive Dashboard

Interactive Dash + Plotly dashboard analysing 435K+ flights from JFK, EWR & LGA in 2023.

## Quick Start

```bash
git clone https://github.com/coffeesanti/nyc-flights-dashboard.git
cd nyc-flights-dashboard
pip install -r requirements.txt
python app.py
```

Then open **http://localhost:8050** in your browser.

## Pages

- **Route Map** — Geographic visualization of flight routes from NYC airports
- **Airline Performance** — Compare airlines by flights, delays, and cancellations
- **Delay Deep Dive** — Analyse departure delay patterns by airport, airline, and month
- **Fleet & Speed** — Explore aircraft types, manufacturers, and average speeds

## Requirements

- Python 3.9+
- ~512MB RAM (flights.csv is 38MB)
